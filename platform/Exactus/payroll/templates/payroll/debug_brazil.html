CRITICAL ISSUE FOUND!

No PD codes found for Brazil!

This is definitely why payroll shows zero.
PD codes define earnings (salary, bonuses) and deductions (taxes, INSS). Without them, the system has nothing to calculate.

Quick Fix - Create Brazil PD Codes:
  1. Go to Django Admin: /admin/payroll/pdcode/
  2. Click "Add PD Code"
  3. Create these essential codes:
    • SALARY_BR - Basic Salary (is_earning: ✓)
    • BONUS_BR - Bonus (is_earning: ✓)
    • ALLOWANCE_BR - Allowance (is_earning: ✓)
    • INSS_BR - Social Security (is_deduction: ✓)
    • IRRF_BR - Income Tax (is_deduction: ✓)
  4. Make sure to select "Brazil" as the country
