File Format Instructions
Required Fields:
  • pdcode_code - Unique PD code (e.g., "BASIC", "TAX")
  • pdcode_name - Display name (e.g., "Basic Salary", "Income Tax")
Optional Fields:
  • pdcode_description - Description
  • pdcode_status - "Visible" or "Hidden"
  • pdcode_account, pdcode_map_code, pdcode_gl_account - Numeric codes
  • pdcode_frequency - "Recurring" or "Non-recurring"
  • pdcode_type - "Regular" or "Irregular"
  • pdcode_class - "Standard" or "Statutory"
  • pdcode_category - "Payment", "Deduction", "Notional", etc.
  • Boolean flags: pdcode_taxable, pdcode_pensionable, etc. (use True/False)
  • pdcode_categorytype - "Bracketable", "Prorational", etc.
Boolean Values:

Use True, False, 1, 0, Yes, or No for boolean fields.
